01. 제어문 사용하기

if문

특정 조건을 만족하면 다음 문장을 실행해달라는 의미입니다.

number = 3
if number > 2:
    print('X는 2보다 큽니다.')

# result
'X는 2보다 큽니다.'

for문

특정 배열을 반복해달라는 의미입니다.

words = ['apple', 'banana', 'orange']
for word in words:
    print(word)

# result
apple
banana
orange

range() 함수

어떤 숫자만큼 반복하길 원하는 경우 사용할수있습니다.

for i in range(5):
    print(i)

# result
0
1
2
3
4
5

특정 숫자부터 시작해서 반복해달라고할경우 다음과같이 인자값을 2개를 줍니다.
이때 첫번째 숫자는 포함하고 마지막 숫자는 포함하지 않습니다.

for i in range(3, 5):
    print(i)

# result
3
4

break문

for문을 순회하다가 break문을 만나면 즉시 for문을 빠져나옵니다.

for i in range(5):
    if i == 3:
        break
    print(i)

# result
0
1
2

continue문

for문을 순회하다가 continue문을 만나면 이후 문장을 실행하지않고 다음 순회로 넘어갑니다.

for i in range(5):
    if i == 3:
        continue
    print(i)

# result
0
1
2
4

pass문

아무것도 하지 않습니다.
굳이 pass문을 작성하는 이유는 에러가 나는 것을 방지해줍니다.
나중에 배울 클래스에서 주로 사용됩니다.

while True:
    pass
