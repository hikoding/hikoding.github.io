변수

변수

변수는 값을 할당하여 데이터를 저장할 수 있습니다.

Example

a = 5
b = 'hello'
print(a)
print(b)

Result

5
hello

변수 유형(Type) 선언

파이썬은 어떤 타입의 데이터가 들어갈지 선언할 필요가 없으며,
선언 후 다른 타입으로 변경할 수도 있습니다.

Example

a = 5
a = 'Five'
print(a)

Result

Five

문자열(String)

문자 여러개가 나열된것을 문자열이라고 합니다.
홑따옴표(작은따옴표), 쌍따옴표(큰따옴표) 모두 사용 가능합니다

a = "hello"
a = 'hello'

변수 대소문자

변수이름은 대소문자를 구분합니다.
다음과 같이 두개의 변수가 생성됩니다.
Example

a = 3
A = 4
print(a)
print(A)

Result

3
4

변수 유형(Type) 확인

type()은 변수의 데이터 유형을 확인할 수 있습니다.

Example

a = 5
b = 'hello'
print(type(a))
print(type(b))

Result

<class 'int'>
<class 'str'>

형변환(Casting)

변수의 타입을 변경할 수 있습니다.
str() 은 문자, int()는 숫자, float() 은 실수로 변경하는 함수입니다.

a = str(5)
b = int('5')
c = float(5)
print(a, type(a))
print(b, type(b))
print(c, type(c))
